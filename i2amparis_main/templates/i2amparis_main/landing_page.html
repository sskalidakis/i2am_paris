Services

Choose among several types of analyses

Region Based Analysis

Region Based Analysis offers the ability to analyse, visualise and compare data from several regions (or worldwide) for a specific sector, comparing the outputs of the selected models.

Sector Based Analysis

Sector Based Analysis offers the ability to analyse, visualise and compare data from several different sectors such as Energy, Transportation, Building etc. for a specific region, comparing the outputs of the selected models.

Policy Question Based Analysis

Policy Question Based Analysis offers the ability to analyse, visualise and compare data from a selected region giving an insight on selected scenarios.
